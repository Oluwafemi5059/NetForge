# Networking Tools

A starter Python package for networking utilities.

## Structure

- `src/networking_tools/` — package source
- `tests/` — unit tests

## Getting Started

1. Create a virtual environment
   ```bash
   python -m venv .venv
   .venv\Scripts\activate
   ```
2. Install the package in editable mode
   ```bash
   pip install -e .
   ```
3. Run tests
   ```bash
   python -m pytest
   ```
